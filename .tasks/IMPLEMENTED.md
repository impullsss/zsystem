# ZSystem — Реализованные механики

Последнее обновление: 2026-03-24

---

## Боевая система

### Броски и попадание

- **d100 roll-under** — бросок под значение навыка
- **Режимы броска** — обычный, публичный, только ГМ (слепой)
- **Расчёт шанса попадания** — базовый навык + модификаторы атаки + штраф части тела + укрытие + дальность + помеха (токены между) + уклонение цели
- **Слот-машина в чате** — визуальное отображение результата броска
- **Штраф слепоты/головокружения** — отображается в подсказке HUD прицела

### Прицеливание

- **Ползунок прицеливания** — накапливаемый бонус (+N% за шаг, настраивается в настройках)
- **Конусное прицеливание** — PIXI-конус (60°, длина = дальность оружия) следует за мышью; все цели в конусе подсвечиваются с шансом попадания; ЛКМ — выстрел, ПКМ — отмена
- **Конус остаётся открытым** после выстрела — закрывается при нехватке AP, патронов или ПКМ
- **Визуализация шанса** — HUD с процентом попадания обновляется в реальном времени
- **Штраф к точности по части тела** — Голова -40%, Конечности -20%, Торс без штрафа

### Части тела

- Голова, Торс, Левая рука, Правая рука, Левая нога, Правая нога
- Конечности могут уходить в отрицательные значения HP

### Урон

- **Типы урона** — blunt, slashing, piercing, ballistic, fire
- **STR-бонус в ближнем бою** — автоматически добавляется к урону; двуручное оружие: STR × 2
- **Хедшоты** — попадание в голову: ×2 урону после DR и сопротивления
- **Скрытные атаки** — атака из стелса: ×2 урону (стелс не снимается)
- **Трассер** — визуальная линия выстрела между атакующим и целью
- **Перки влияют на урон** — через `PerkLogic.onApplyDamage()`
- **Горение** — игнорирует DR, но учитывает сопротивление огню

### Покрытие и дальность

- **Расчёт укрытия** — проверка пересечения луча с объектами сцены
- **Штраф дальности** — зависит от параметров оружия
- **Проверка помех** — токены между атакующим и целью снижают точность

### Оружие (лист предмета)

- Выбор руки (левая/правая) при экипировке одноручного оружия — бесплатно первый раз, -1 AP при смене
- Смена руки кликом по бейджу `[П]/[Л]` на листе персонажа
- -1 AP при надевании оружия
- **Варианты атак** — таблица с колонками: Название, AP, Пт., Урон, Точн., **Шум** (модификатор к базовому шуму)

## Статусы и эффекты

### Травмы (физические, постоянные)

| Статус | Эффект |
|--------|--------|
| Травма головы | -50% INT/PER, -15 к Науке/Механике/Медицине, бросок на потерю AP в начале хода |
| Травма торса | -1/3 AGI/VIG, движение наносит 1d5 урона |
| Сломана рука (лев./прав.) | -10% к точности, атаки стоят +1 AP на сломанную руку |
| Сломана нога (лев./прав.) | -10 уклонение, каждый шаг +1 AP за каждую сломанную ногу |
| Без сознания | AP = 0 |

### Временные статусы

| Статус | Эффект |
|--------|--------|
| Кровотечение | 4d4 HP урона каждый ход (истинный урон, игнорирует DR) |
| Сбит с ног | Уклонение -50%, встать стоит 3 AP |
| Головокружение | -50% к шансу попадания |
| Слепота | Ползунок прицеливания заблокирован, обзор токена сужается до 1.5 клетки |
| Горение | 1d6 урона по каждой конечности каждый ход |
| Отравление | 1d6 HP урона каждый ход |
| Инфекция | Скрытый статус, смерть через 3 дня (стадии 1-3) |
| Ранен | -2 макс AP |
| Утомление | Стакается до 5×: -10 макс HP, -1 макс AP, -5% навыков |
| Перегруз | Уклонение -50%, движение +1 AP |
| Скрытность | ×2 урон из стелса; +1 AP к движению; счётчик шагов; бросок при входе в зону зомби |
| Невидимость | Токен скрыт на сцене |

### Паника (стадии)

- **Тревога** — +1 AP к атакам, +5 уклонение
- **Паника** — -10% к точности, +2 AP к атакам, +10 уклонение
- **Срыв** — AP = 0 (потеря хода)
- `fullHeal()` корректно снимает все стадии паники

### Инфекция (стадии)

- Стадия 1 — инкубация, нет эффектов
- Стадия 2 — симптомы: -1 STR/AGI/VIG
- Стадия 3 — кризис: -3 STR/AGI/VIG, -3 макс AP

## Система скрытности и обнаружения

- **Статус "Скрытность"** — активируется через HUD или лист
- **Счётчик шагов** — `floor(Stealth/10)` шагов, тратится при каждом движении (везде); закрепляется над токеном через PIXI-дочерний элемент
- **Бросок d100** при входе в зону обнаружения зомби — успех/провал в чат ГМа
- **Кольца обнаружения** (только ГМ) — оранжевый круг PER-радиуса вокруг каждого зомби
- **При провале** — снимается статус скрытности, красная точка на месте обнаружения, надпись "ОБНАРУЖЕН!" 6 сек., пауза игры
- **Восстановление шагов** — вне колец при исчерпании, внутри кольца при исчерпании → новый бросок

## Система шума

- Каждое действие с оружием генерирует шум (базовый + модификатор атаки)
- Шум затухает вдвое каждый раунд боя
- **NoiseManager HUD** — перетаскиваемое окно, ручной ввод, +/-, сворачивание
- Позиция и состояние сохраняются в localStorage

## Персонажи

### Атрибуты

STR (Сила), AGI (Ловкость), VIG (Живучесть), PER (Восприятие), INT (Интеллект), CHA (Харизма)

### Навыки (10 штук)

Рукопашный, Стрельба, Наука, Механика, Медицина, Дипломатия, Лидерство, Выживание, Атлетика, Скрытность

### Вторичные параметры

HP, AP (очки действия), уклонение, вес (переносимый/максимальный), инициатива, очки перков

### Типы акторов

- **survivor / npc** — полноценный персонаж
- **zombie** — враг с упрощёнными статами (с опциональной рандомизацией STR/AGI/VIG)
- **shelter** — база/убежище (управление ресурсами)
- **container** — контейнер с лутом (ловушки, замки, скрытые тайники)
- **harvest_spot** — точка сбора ресурсов
- **vehicle** — транспорт

## Предметы

### Типы

weapon, armor, consumable, ammo, resource, medicine, food, materials, luxury, misc, upgrade, project, perk

### Оружие

- Хват: одноручное / двуручное
- Выбор руки экипировки; смена руки -1 AP
- Варианты атак с полем модификатора шума
- Метательное оружие (шаблоны: круг/конус)

### Броня

- DR по типу урона (blunt/slashing/piercing/ballistic/fire)
- Карта покрытия частей тела

### Перки

- **Runner** — каждый N-й шаг бесплатен
- **Butcher** — бонус урона по кровоточащим целям

## UI / HUD

### Player HUD

- Появляется при выборе своего survivor-токена
- Перетаскивается, позиция сохраняется в localStorage
- Кнопки: Скрытность (toggle), Встать (если Prone), Перезарядить, Конец хода
- Динамически обновляется при изменении AP/HP/эффектов

### Noise HUD

- Перетаскиваемое окно, сворачивается
- Ручной ввод, кнопки +/-

## Убежище

- Ресурсы: еда, топливо, запчасти, антибиотики, роскошь
- Мораль жителей
- Жители: drag-and-drop назначение на работу
- Улучшения базы (строительство)
- Проекты (крафт/исследование)
- Защита базы (суммируется из завершённых улучшений)
- Ежедневное потребление ресурсов с голодом при нехватке еды

## Транспорт

- Расход топлива = расстояние ÷ MPG
- Расчёт времени в пути
- Блокировка движения при недостатке топлива
- Лог путешествия в чате

## Создание персонажа

- Отдельное окно приложения
- Распределение очков атрибутов (SP) и очков перков (PP)

## Архитектурные особенности

- **GM-модель** — вся изменяющая состояние логика выполняется только на клиенте ГМ через `socketlib`
- **Кнопка отмены урона** — в контекстном меню чат-сообщения
- **Перенос предметов** — drag & drop с проверкой расстояния
- **Рандомизация зомби** — настройка в параметрах системы
- **Глобальная карта** — флаг сцены, отдельная логика движения через TravelManager
