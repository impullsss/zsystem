**Статус:** backlog

**Создана:** 2026-03-23

## Описание

Скрипт должен автоматически прибавлять значение атрибута STR к урону холодного оружия.
Для двуручного оружия — STR × 2.

## Критерии готовности

- [ ] Одноручное холодное оружие: итоговый урон += STR
- [ ] Двуручное холодное оружие: итоговый урон += STR × 2
- [ ] Огнестрельное/метательное оружие — без изменений
- [ ] Значение STR берётся из актора-атакующего

## Технические заметки

- Файл: `src/module/dice.js` — функция `_executeAttack()`, блок расчёта finalDmg (≈ строка 561)
- Хват оружия: `item.system.hands` — значения `"1h"` (одноручное) или `"2h"` (двуручное)
- Тип оружия: `item.system.weaponType` — значения `"melee"` или `"ranged"`
- Значение STR: `actor.system.attributes.str.value`
- Логика: если `weaponType === "melee"` → добавить STR; если `hands === "2h"` → добавить STR × 2 вместо STR
