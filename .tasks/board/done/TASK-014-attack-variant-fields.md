# TASK-014: Улучшение вариантов атак (шум + выравнивание)

**Статус:** backlog
**Приоритет:** medium
**Создана:** 2026-03-23

## Описание

В блоке "Варианты атаки" на листе предмета не хватает поля шума.
Также нужно выровнять колонки (Название, AP, Пт., Урон, Точн., Эффекты) — сейчас они съезжают.

## Желаемое поведение

- В каждом варианте атаки добавить поле **Шум** — значение добавляется/вычитается из базового шума оружия
  - Пример: оружие шум 30, атака "Выстрел дробью" шум +0, атака "Навскидку" шум -10 → итог 20
  - Знак может быть отрицательным (глушитель-режим, осторожный выстрел)
- Колонки таблицы вариантов атаки выровнять через фиксированные ширины или grid
