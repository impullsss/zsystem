## Описание
Реализовать формулу агро-радиуса для зомби с учётом скрытности игрока.

## Формула
```
Радиус Агро = PER врага (клетки) - floor(Stealth игрока / 10)
```

### Пример
- Зомби PER = 5 → базовый радиус = 5 клеток
- Игрок Stealth = 40 → снижение на 4
- Итог: зомби замечает с 1 клетки

При провале скрытности — радиус стандартный (без снижения).

## Критерии готовности
- [ ] Зомби получает поле "Detection Radius" (или вычисляется из PER)
- [ ] Каждый ход (или по хуку движения) проверяется расстояние между скрытным игроком и зомби
- [ ] Если дистанция > радиуса агро — зомби не реагирует
- [ ] Если дистанция ≤ радиусу — зомби переходит в статус `alerted`
- [ ] При провале проверки скрытности (или отсутствии статуса Stealth) — радиус = PER как есть

## Технические заметки
- Можно интегрировать в существующую систему шума (`noise.js`)
- Или отдельный хук на движение токена (`preUpdateToken` / `updateToken`)
- Проверка скрытности: `stealthActor.hasStatusEffect("stealth")` + `system.skills.stealth.value`
- Визуализация: отображать радиус как circle (аналогично noise debug mode)
