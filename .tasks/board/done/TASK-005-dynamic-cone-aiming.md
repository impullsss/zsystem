## Описание

Режим прицеливания в котором игрок сам наводится на цель через визуальный конус.
Конус появляется перед токеном атакующего и при наведении на врага показывает вероятность попадания в него.

## Желаемое поведение

- При активации режима перед токеном рисуется конус (угол = разброс оружия?)
- При наведении конуса на токен врага — появляется подсказка с шансом попадания
- Если несколько врагов в конусе — показывать шансы для каждого
- ЛКМ по цели = выстрел, ПКМ = отмена (как сейчас в Smart Aiming)

## Вопросы для уточнения перед реализацией

- Конус заменяет текущий Smart Aiming или работает параллельно?
- Угол конуса фиксированный или зависит от оружия/навыка?
- Нужно ли учитывать дальность (конус обрезается по range оружия)?
- Дробовик — широкий конус с несколькими целями одновременно?

## Технические заметки

- Базируется на текущем `AimingManager` в `src/module/dice.js`
- Конус рисуется через PIXI.Graphics (уже используется для линии и круга)
- Шанс попадания через существующую `_calculateHitChance()`
- Может потребоваться отдельный класс `ConeAimingManager`
