## Описание
Когда игрок выбирает своего персонажа на сцене — в левом нижнем углу появляется компактный HUD с быстрыми действиями, доступными системе.

## Примеры кнопок
- Встать в скрытность / Выйти из скрытности
- Встать (если Prone)
- Перезарядить оружие
- Конец хода (pass turn)
- Возможно: статусы одним взглядом (AP, HP, иконки активных эффектов)

## Критерии готовности
- [ ] HUD появляется при выборе токена типа `survivor`
- [ ] Кнопки динамически активны/неактивны в зависимости от текущего состояния (есть ли уже статус, достаточно ли AP)
- [ ] HUD скрывается при снятии выделения
- [ ] Все действия через существующие методы актора (не дублировать логику)

## Технические заметки
- Тригер: `controlToken` hook (`controlled: true/false`)
- Рендер: Application или простой div, добавляемый в `#ui-bottom`
- Не показывать игрокам чужой HUD — проверять `game.user.isGM || token.isOwner`
