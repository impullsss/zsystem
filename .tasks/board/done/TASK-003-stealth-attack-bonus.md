# TASK-003: Скрытные атаки

**Статус:** backlog
**Приоритет:** medium
**Создана:** 2026-03-23

## Описание

Если атакующий находится в режиме Скрытности (stealth), урон от атаки умножается на 2.
Скрытность после атаки НЕ снимается автоматически.

## Критерии готовности

- [ ] Проверка статуса stealth у атакующего перед расчётом урона
- [ ] Множитель ×2 применяется к финальному урону
- [ ] В чат-сообщении отображается пометка "Скрытная атака"
- [ ] Скрытность после атаки остаётся на акторе

## Технические заметки

- Файл: `src/module/dice.js` — функция `_executeAttack()` (строка 446)
- Статус stealth: `actor.statuses.has("stealth")` — определён в `constants.js` с id `"stealth"`
- Множитель применяется к `finalDmg`
- В описании статуса уже написано "Атака из стелса наносит х2 урон" — нужно это реализовать
