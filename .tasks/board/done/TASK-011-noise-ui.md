## Описание
Улучшить HUD окно шума: дать возможность перемещать его по экрану, скрывать и вводить значение вручную.

## Критерии готовности
- [ ] Окно шума можно перетаскивать (drag) по экрану, позиция сохраняется в localStorage или флаге
- [ ] Кнопка скрыть/показать окно (или правый клик по нему)
- [ ] Поле ввода числа вместо (или рядом с) кнопками +/− для ручного задания уровня шума
- [ ] Всё это работает только у ГМа (окно шума уже ГМ-only)

## Технические заметки
- Файл: `src/module/noise.js` — там рендерится HUD
- Перетаскивание: jQuery draggable или нативный Foundry `Application.setPosition()`
- Сохранение позиции: `game.user.setFlag("zsystem", "noiseHudPos", {left, top})`
- Ручной ввод: заменить span на `<input type="number">` с `change` хуком
