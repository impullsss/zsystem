# TASK-021 — Расширение системы перков

**Статус:** backlog
**Приоритет:** low
**Размер:** L

## Цель

Добавить новые перки и сделать систему перков более гибкой.

## Текущее состояние

Реализовано только два перка через `PerkLogic`:
- **Runner** — каждый N-й шаг бесплатен
- **Butcher** — бонус урона по кровоточащим целям

## Идеи для новых перков

### Боевые
| Перк | Эффект |
|------|--------|
| **Iron Stomach** | Иммунитет к отравлению |
| **Quick Hands** | Перезарядка стоит на 1 AP меньше |
| **Executioner** | Бонус урона +20% когда у цели <25% HP |
| **Tough** | +10 макс HP |

### Скрытность
| Перк | Эффект |
|------|--------|
| **Ghost** | Шаги не расходуют счётчик вне боя |
| **Ambush Expert** | Бонус из стелса ×3 вместо ×2 |

### Выживание
| Перк | Эффект |
|------|--------|
| **Field Medic** | Лечение через аптечки +25% |
| **Scavenger** | Находит +1 предмет при обыске контейнеров |

## Технически

- Перки-предметы с `subtype` → хук в `PerkLogic.onXxx()`
- Пассивные перки через ActiveEffects (уже работает)
- Активные — через `PerkLogic.checkPerk(actor, "perkId")`

## Требует уточнения

Приоритет перков и их баланс — согласовать с правилами в `GAME_RULES.md`.
