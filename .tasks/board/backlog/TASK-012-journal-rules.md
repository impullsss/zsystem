## Описание
Создать Journal Entry который автоматически подгружается в Foundry при старте системы.
Внутри — основные правила игры для игроков: статусы, травмы, боевые механики, атрибуты.

## Критерии готовности
- [ ] Journal Entry создаётся программно через `CONFIG` или `Hooks.once("ready")`
- [ ] Проверка — не создавать повторно если уже существует (по ID или флагу)
- [ ] Контент берётся из `GAME_RULES.md` (адаптировать под Foundry Journal формат)
- [ ] Доступен всем игрокам (права на чтение)
- [ ] Сортируется первым в списке журналов или в отдельной папке "Правила"

## Технические заметки
- Хук: `Hooks.once("ready", ...)`
- API: `JournalEntry.create({ name, content, ownership })`
- Флаг для проверки уникальности: `game.settings.get("zsystem", "rulesJournalId")`
- HTML-контент можно генерировать из константы в `constants.js`
