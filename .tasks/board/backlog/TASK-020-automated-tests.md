# TASK-020 — Автотесты: базовая инфраструктура

**Статус:** backlog
**Приоритет:** medium
**Размер:** L (большая задача, разбить на части при старте)

## Цель

Настроить тестирование чистой логики (без FoundryVTT окружения), чтобы не тестировать руками каждый раз.

## Что тестируется (первый приоритет)

После рефакторинга TASK-018/019 функции-хелперы не зависят от Foundry API и тестируемы изолированно:

| Функция | Тест |
|---------|------|
| Расчёт урона с DR | `applyDamage(actor, 20, "ballistic", "torso")` при DR=5 → 15 |
| Хедшот ×2 | `applyDamage(..., "head", true)` → урон удваивается после DR |
| Горение игнорирует DR | `applyDamage(..., "torso", false, true)` → DR=0 |
| Стадии паники | `checkPanic(actor, 30)` при HP=50 → стадия паники |
| Расчёт навыков | `prepareBaseData` → навык = base + points + mod |
