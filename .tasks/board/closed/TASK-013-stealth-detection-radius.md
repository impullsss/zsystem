# TASK-006: Система обнаружения — Радиус Агро и Скрытность

**Статус:** backlog
**Приоритет:** medium
**Создана:** 2026-03-23

## Описание

Каждый враг имеет Радиус Обнаружения (Detection Radius).
Навык Скрытность (Stealth) игрока уменьшает этот радиус.

## Формула

```
Радиус Агро = PER врага (в клетках) - (Stealth игрока / 10)
```

**Пример:**
- Зомби PER = 5 → базовый радиус = 5 клеток
- Игрок Stealth = 40 → снижение на 4 клетки
- Итог: зомби замечает игрока только с 1 клетки

## Поведение при провале скрытности

Если игрок провалил бросок скрытности → радиус становится стандартным (без снижения).
